# SQL-анализ продаж интернет-магазина (PostgreSQL)

12 бизнес-вопросов к базе интернет-магазина на чистом SQL - от простых
агрегатов до оконных функций. Цель - получить из данных выводы и рекомендации.

## База данных
Нормализованная схема из 4 связанных таблиц:
```
```
| Таблица | Описание | Строк |
|---|---|---|
| `customers` | клиенты: страна, сегмент, дата регистрации | 300 |
| `products` | товары: категория, цена | 40 |
| `orders` | заказы: дата, статус (completed / cancelled / returned) | 4 000 |
| `order_items` | позиции заказа: количество, цена, скидка | 10 049 |

Базу можно воссоздать одним скриптом - [`build_db.sql`](build_db.sql).

## SQL-приёмы в проекте
`JOIN` (несколько таблиц) · `GROUP BY` / `HAVING` · агрегаты (`SUM`, `COUNT`) ·
`CTE` (`WITH`, многослойные) · **оконные функции** (`LAG`, `RANK`, `ROW_NUMBER`, `AVG`/`SUM` `OVER`) · фильтрация окна через CTE ·
`PARTITION BY` · `WHERE`,`HAVING` · сортировка · `LIMIT`.

## Бизнес-вопросы и выводы
Все запросы - в [`queries.sql`](queries.sql).

| # | Вопрос | Вывод |
|---|---|---|
| 1 | Общая выручка | 6.36M (грязная, по всем позициям) |
| 2 | Заказы по статусам | ~19% заказов отменены/возвращены - потерянная выручка |
| 3 | Выручка по категориям | Electronics ≈ 47% всей выручки |
| 4 | Выручка по месяцам | пики в ноябре–декабре (предновогодняя сезонность) |
| 5 | Топ-10 клиентов | топ-клиент ≈ 0.7% выручки → нет зависимости от «китов» |
| 6 | Реальная выручка (completed) | 5.12M; ~1.24M (20%) теряется на отменах |
| 7 | Прирост месяц-к-месяцу (`LAG`) | +315K в ноябрь, −167K в январь |
| 8 | Рейтинг товаров в категории (`RANK` + `PARTITION BY`) | электроника доминирует |
| 9 | Доля категорий (`SUM() OVER ()`) | топ-2 категории = 72% выручки (концентрация) |
| 10 | Лояльные клиенты (`HAVING`) | много клиентов с 5+ заказами - база для удержания |
| 11 | Топ-3 товара в категории (`ROW_NUMBER` + фильтр в CTE) | лидеры продаж внутри каждой категории |
| 12 | Товары выше среднего по категории (`AVG() OVER` + фильтр) | продукты-передовики относительно своей категории |
